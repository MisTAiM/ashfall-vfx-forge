# ASHFALL VFX FORGE v3.0

Full-scale HTML5 Canvas VFX animation platform for [Ashfall Idle](https://ashfall-idle.vercel.app).

## Features

- **44 VFX Effects** across 7 categories
- **10 Particle Types** with pooled physics engine (400 cap)
- **8 Attack Animations** — Slash, Thrust, Projectile, Magic Bolt, AOE, Crit, Multi-Hit, Special
- **6 Combat VFX** — Damage Numbers, Hit Impact, Heal, Buff, Debuff, Block
- **6 Auras** — Radial Glow, Flame, Frost, Shadow, Lightning, Holy
- **6 Transforms** — Float, Spin, Pulse, Wobble, Bounce, Orbit
- **6 Post-FX** — Bloom, Chromatic Aberration, Scanlines, Vignette, Color Grade, Screen Shake
- **12 VFX Presets** — One-click effect combos
- **33 Item Database** — OSRS-style equipment slots with tier/element classification
- **Code Export** — Generate JS config objects for game integration
- **Frame Export** — Download canvas as PNG
- **Real-Time Canvas** — 60fps DPR-aware rendering
- **Full Wiki** — Documentation at `/wiki.html`

## Stack

- React 18 (CDN)
- HTML5 Canvas
- SVG Icons (zero emoji)
- Rajdhani + Share Tech Mono typography

## Deployment

Static site — deploys directly to Vercel with zero config.
